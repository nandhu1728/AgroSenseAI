AgroSense AI

AgroSense AI is a smart agriculture prototype developed to help farmers obtain fertilizer suggestions using soil parameters through artificial intelligence.

This system simulates soil sensor values such as **NPK values and soil moisture** and gives a smart fertilizer recommendation.

This project aims to help farmers make better decisions in their farming processes.

###  Features

• Soil parameter simulation (N, P, K values)
• Moisture level simulation
• AI-based fertilizer suggestion
• Language change (Tamil / English)
• Recent search history
• Simple and easy-to-use interface

### How It Works

1. User clicks on **Simulation Button**
2. System generates sample values
3. AI analyzes values
4. Shows fertilizer recommendation

This prototype is a demonstration of how artificial intelligence can be applied in precision agriculture.
